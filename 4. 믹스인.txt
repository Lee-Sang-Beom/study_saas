1. 믹스인 : 사이트 전체에서 재사용할 스타일 [그룹]을 정의하는 도구
 - 변수는 재사용할 속성을 정의하는 도구

2. 사용방법 
 - @mixin 키워드 사용
 - @include로 호출

'''
@mixin my-font{
    font-family: sans-seif;
    font-size: 32px;
    font-style: normal;
    font-weight: 500;
    color: orange;
}

h1{
    @include my-font;
}

p{
    @include my-font;
}
'''

3. 믹스인 인자
 - 함수에 전달하는 파라미터처럼 사용

'''
@mixin my-font($font-color, &bg-color: yellow) {
    font-family: sans-seif;
    font-size: 32px;
    font-style: normal;
    font-weight: 500;
    color: $font-color;
    background-color: &bg-color;
}

h1{
    @include my-font(red); // h1의 배경색은 자동으로 yellow로 적용된다.
}

p{
    @include my-font(blue, red);
}
'''
